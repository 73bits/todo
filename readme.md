# todo

add a new todo specify title.
```
todo -add <string>
```

specify a todo by index to delete. (default -1)
```
todo -del <int>
```

edit a todo by index and specify a new title. id:newtitle
```
todo -edit <string>
```

list all todos
```
todo -list
```

specify a todo by index to toggle. (default -1)
```
todo -toggle <int>
```
